usePlaceCage

React hook for generating a many wonderful instance of Nicolas Cage via placecage.com

Examples

Single Nick

const nickSingle = usePlaceCage(400, 300);

Nick with his clones

const nickClones = usePlaceCage({
  width: 300,
  height: 300,
  count: 5
});

Nick with friends

const nickFriends = usePlaceCage([
  {
    width: 100,
    height: 200,
    count: 1
  },
  {
    width: 110,
    height: 200,
    count: 1
  },
  {
    width: 120,
    height: 200,
    count: 1
  },
  {
    width: 130,
    height: 200,
    count: 1
  },
  {
    width: 140,
    height: 200,
    count: 1
  },
  {
    width: 150,
    height: 200,
    count: 1
  }
]);
